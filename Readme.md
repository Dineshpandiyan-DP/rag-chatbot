# RAG Knowledge Assistant

An end-to-end Retrieval-Augmented Generation (RAG) application that enables users to query documents using natural language. The system ingests documents, creates embeddings, stores them in a vector database, retrieves relevant context, and generates accurate responses using Large Language Models (LLMs).

## Features
- Document ingestion
- Text chunking
- Embedding generation
- Vector similarity search
- Context-aware response generation
- FastAPI backend
- Modular architecture
